## О проекте

**TLVanced** — это десктопное приложение на **WPF** для Windows: аккуратное окно с быстрым доступом к двум полезным сценариям — **правка движкового конфига** и **правка HUD-лейаута** под ваши задачи. Всё делается локально на машине, без серверов и аккаунтов.

> Проект создан величайшим погромистом и стримером ТАНГЕНСОМ и **никак не связан** с криворуким издателем или разработчиком игры. Используйте на свой страх и риск; перед изменениями имеет смысл сделать резервные копии файлов.

## Возможности

### 1. UE4 Конфиг

Добавляет и редактирует параметры запуска **Unreal Engine 4** в конфиге клиента **Throne and Liberty** — через файл:

`%LocalAppData%\TL\Saved\Config\WindowsNoEditor\Engine.ini`

В отдельном окне вы получаете **таблицу**: имя cvar, текущее значение (если в конфиге пусто — показывается `*`), подсказки по допустимым значениям и краткое описание. Удобно **оптимизировать FPS**, убирать тяжёлые эффекты и **тонко донастраивать** картинку под своё железо — от motion blur и тумана до Niagara, травы, foliage и атмосферы неба.

- Читается секция **`[/Script/Engine.RendererSettings]`** (без учёта регистра).
- Если секции ещё нет — она будет создана при сохранении.
- Строки, которых нет в вашем пресете приложения, при сохранении **не затираются** — правятся только известные ключи и добавляются новые.

---

### 2. UI Фикс

Убирает с экрана два навязчивых HUD-элемента — **диалог с NPC** и **диалог с Амитоем** — за счёт правки лейаута Azulejo:

`%UserProfile%\Documents\TL\UserHUD\Main.azj`

Для компонентов с ключами **145** и **128** выставляются: выравнивание **LeftTop**, масштаб **0**, смещение **(-1000, -1000)** — фреймы уезжают за пределы видимой области и перестают мешать.

- Кнопка **красная**, если `Main.azj` не найден.
- Текст кнопки **зелёный**, если оба компонента уже «убраны» по этим правилам.

Чтобы это заработало вам нужно экспортировать профиль, который потом будет отредактирован TLVanced
- Заходим в игру
- Открываем HUD Редактор
- Слева внизу жмём кнопку Экспорта (Стрелочка)
- В окне ввода названия пишем - Main! ОБЯЗАТЕЛЬНО!
- В TLVanced нажимаем UI Фикс
- Если надпись загорелась зелёным, значит всё окей!
- В игре, в HUD Редакторе нажимаем кнопку Изменить
- Справа от названия файла будет стрелка
- Жмём загрузить
- Кайфуем

## Сборка

```bash
git clone https://github.com/tnhgnz/TL-VANCED.git
cd TLVanced/TLVanced
dotnet restore
dotnet build -c Release
```

Готовый exe появится в `TLVanced/bin/<Platform>/Release/net9.0-windows/`.

> В репозитории используется локальная ссылка на **Wpf.Ui** (`Libs\Wpf.Ui.dll`) — убедитесь, что файл есть в дереве проекта, либо замените ссылку на NuGet-пакет под вашу среду.
